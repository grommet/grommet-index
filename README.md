# [Maintenance Mode] grommet-index
The Grommet team has placed grommet-index in maintenance mode. Although we are not actively 
adding to this repository, we are still accepting pull requests from the community to maintain your project dependencies. 

We are doing some great things with [grommet-addons](https://github.com/grommet/grommet-addons) and promote looking in to migrating your project to use
grommet-addons as grommet-index will have limited support. We invite you to check it out!

### Purpose
UI and example server for searching, filtering, and displaying mass quantities of items.

### Documentation

See http://index.grommet.io
